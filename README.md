## Installation

### Requirements

- NodeJs
- NPM ( comes with Nodejs )
- VS Code ( Optional )
- Browser
- Operating System
- NPM Command : npm install

## Setup

1. Install NodeJs :-
   https://nodejs.org/en/download/

2. Command :-

```bash
$ npm install

```

```bash
$ npm start
```
